# Abstract Em-Dash Convention

## Zero-tolerance rule

Em-dashes are not acceptable in JCP abstracts. A survey of 4 2024 JCP PINN papers found zero em-dashes in any abstract.

If an em-dash appears in a direct quotation that cannot be paraphrased, flag it for the author to rephrase.

## Chinese abstract — fix patterns

| —— usage | Fix | Example |
|----------|-----|---------|
| Elaboration | `。具体而言，` or `：` (short only) | `提出新框架——它通过低秩约束正则化` → `提出新框架，它通过低秩约束正则化` |
| Result/conclusion | `；` or `，且` | `降低45%——方差也降` → `降低45%；且方差也降` |
| Contrast | `，而` | `1D竞争——2D系统性超越` → `1D竞争，而2D系统性超越` |

## English abstract — fix patterns

| — usage | Fix | Example |
|---------|-----|---------|
| Elaboration | colon or "specifically" | "LoRA excels — it reduces error by 45%" → "LoRA excels, reducing error by 45%" |
| Result | "which" / ", yielding" | "uses 23% parameters — outperforms full FT" → "uses 23% parameters yet outperforms full FT" |
| Parenthetical | commas | "PINNs — unlike FDM — are mesh-free" → "PINNs, unlike FDM, are mesh-free" |
