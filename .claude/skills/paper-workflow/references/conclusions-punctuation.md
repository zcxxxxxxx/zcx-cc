# Conclusions Em-Dash Convention

Conclusions sections are the shortest (~2–5% of the paper) and typically problem-free. If an em-dash appears, apply the same result/conclusion pattern as Discussion and Results.

## Chinese patterns

| —— function | Fix | Example |
|-------------|-----|---------|
| Conclusion | `，因此` / `，说明` | `LoRA以23%参数获胜——PEFT可行` → `LoRA以23%参数获胜，说明PEFT可行` |
| Contrast | `，而` | `1D受限——2D优势明确` → `1D受限，而2D优势明确` |

## English patterns

| — function | Fix | Example |
|------------|-----|---------|
| Conclusion | "suggesting" / "indicating" | "LoRA wins with 23% params — PEFT is viable" → "LoRA wins with 23% params, indicating PEFT is viable" |
| Contrast | "whereas" / "while" | "PEFT wins on accuracy — full FT is cheaper" → "PEFT wins on accuracy, whereas full FT is cheaper" |

No action needed if the conclusions section has zero em-dashes — this is the expected state.
