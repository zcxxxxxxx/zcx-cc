# Redundant Phrases & Wordiness — Quick Reference

## Hollow Lead-ins (delete or rephrase)

| ❌ Redundant | ✅ Concise |
|-------------|-----------|
| It is worth noting that... | (state directly) |
| It should be pointed out that... | (state directly) |
| It can be seen that... | (state directly) |
| It is important to note that... | (state directly) |
| It has been demonstrated that... | (state directly) |
| As a matter of fact | In fact / (omit) |
| In order to | To |
| Due to the fact that | Because |
| In spite of the fact that | Although |
| In the event that | If |
| With the exception of | Except |
| At this point in time | Now / Currently |
| For the purpose of | For / To |
