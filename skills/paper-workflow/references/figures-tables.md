# Figures & Tables

Figures and tables convey data visually/numerically. Choose the right format: **tables** for precise values, **figures** for patterns and trends. **Never use both for the same data.**

## Placement

Two conventions — follow the target journal's Instructions for Authors:

| Convention | Description |
|-----------|-------------|
| **End of manuscript** | Traditional. All tables first (one per page), then all figures. Still required by many journals |
| **Near text** | Modern. Place each table/figure immediately after the paragraph where first cited. Increasingly common |

## Numbering & Citation

- **Sequential numbering** per type: Table 1, Table 2; Figure 1, Figure 2, etc.
- **Every item must be cited** at least once: "as shown in Figure 1" or "(Table 2)"
- Number in the order of first mention

## Captions

| Element | Rule |
|---------|------|
| **Table caption** | **Above** the table. Bold "Table 1." followed by descriptive title. Self-sufficient standalone |
| **Figure caption** | **Below** the figure. Include number, title, and enough detail to interpret without the main text |

## In-text Reference Style

- **Table reference**: "Table 1 lists…" or "…as summarized in Table 1"
- **Figure reference**: "Figure 2 shows…" or "…as illustrated in Figure 2"
- Use **present tense** when referring to figures/tables: "Fig. 2 shows…" (not "showed")
- Do **not** repeat caption content verbatim in the main text — summarize trends instead

## Data Presentation Rules

- **No text–figure redundancy**: If data are in a table/figure, do not list the same numbers in the Results text. Summarize the trend, refer to the exhibit
- **Tables**: Use horizontal rules sparingly (header + bottom only). No vertical rules. Align decimal points. Left-align text
- **Figures**: High resolution, legible fonts, thick enough lines. Label axes with quantity/unit. Include legends. Error bars must specify ±SD or ±SEM
- **Supplementary materials**: Move large tables/extended data to supplementary; include key results in the main text

## Common Mistakes

- Figure resolution too low (journal minimum is typically 300 DPI)
- Table too dense (more than 10 columns or 20 rows → consider splitting or moving to supplementary)
- Caption not self-sufficient (reader shouldn't need to search the main text to understand what a figure shows)
- Missing error bars or failing to define them
- Color-only distinctions that are invisible in grayscale print
- Tables with too many decimal places (round to meaningful precision)
