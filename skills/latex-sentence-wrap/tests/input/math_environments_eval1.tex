\section{Methods}
\label{sec:methods}

Consider the one-dimensional Burgers equation with viscosity $\nu$:
\begin{equation}
\label{eq:burgers}
\frac{\partial u}{\partial t} + u \frac{\partial u}{\partial x} = \nu \frac{\partial^2 u}{\partial x^2}, \qquad x \in [-1, 1],\; t \in [0, 1].
\end{equation}
The PINN loss function is given by $\mathcal{L}(\theta)
= \lambda_f \mathcal{L}_f + \lambda_{ic} \mathcal{L}_{ic} + \lambda_{bc}
\mathcal{L}_{bc}$,
where each term is a mean-squared error over collocation points.
The PDE residual loss is:
\begin{align}
\mathcal{L}_f &= \frac{1}{N_f} \sum_{i=1}^{N_f} \left| \frac{\partial \hat{u}}{\partial t}(x_f^i, t_f^i) + \hat{u} \frac{\partial \hat{u}}{\partial x}(x_f^i, t_f^i) - \nu \frac{\partial^2 \hat{u}}{\partial x^2}(x_f^i, t_f^i) \right|^2, \\
\mathcal{L}_{ic} &= \frac{1}{N_{ic}} \sum_{i=1}^{N_{ic}} \left| \hat{u}(x_{ic}^i, 0) - u_0(x_{ic}^i) \right|^2, \\
\mathcal{L}_{bc} &= \frac{1}{N_{bc}} \sum_{i=1}^{N_{bc}} \left| \hat{u}(-1, t_{bc}^i) - u(-1, t_{bc}^i) \right|^2 + \left| \hat{u}(1, t_{bc}^i) - u(1, t_{bc}^i) \right|^2.
\end{align}
LoRA decomposes the weight update as $\Delta W = \frac{\alpha}{r} B A$,
where $B \in \mathbb{R}^{d \times r}$, $A \in \mathbb{R}^{r \times d}$,
and $r \ll d$~\cite{hu2022lora}.
During adaptation, the forward pass becomes $h = W_s x + \frac{\alpha}{r} B A
x$,
with only $A$ and $B$ updated via gradient descent.
