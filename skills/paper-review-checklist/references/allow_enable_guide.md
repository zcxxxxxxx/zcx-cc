# Allow / Enable — Must Take an Object

Both verbs require a direct object specifying **who** is enabled/allowed to do something.

## Examples
- ❌ "The framework allows to solve PDEs faster."
- ✅ "The framework allows us to solve PDEs faster."
- ❌ "This approach enables to achieve higher accuracy."
- ✅ "This approach enables researchers to achieve higher accuracy."

## Pattern
`X allows/enables [someone/something] to [verb]`

Never write `allows to` or `enables to` without an object.
